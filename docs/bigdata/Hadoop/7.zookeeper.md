# ZooKeeper 分布式协调服务详解

## 1. ZooKeeper 概述

ZooKeeper 是一个分布式协调服务，用于维护配置信息、命名服务、分布式同步和组服务。它为分布式应用提供高性能、高可用、严格顺序访问的协调功能。

### 1.1 ZooKeeper 特性

- **简单**：提供简单的原语集，易于理解和使用
- **高性能**：针对读操作进行了优化，支持高并发
- **高可用**：集群部署，避免单点故障
- **严格顺序访问**：保证更新的顺序性
- **数据一致性**：保证所有客户端看到相同的数据视图

## 2. ZooKeeper 架构

### 2.1 核心组件

- **Server**：ZooKeeper 服务器，提供协调服务
- **Client**：ZooKeeper 客户端，与服务器交互
- **ZNode**：数据节点，存储数据
- **Watcher**：监听器，监听节点变化

### 2.2 架构原理

```java
// ZooKeeper 架构示意
/*
ZooKeeper Ensemble (集群)
├── Leader (领导者)
│   ├── 处理写请求
│   ├── 协调投票
│   └── 状态同步
├── Follower (跟随者)
│   ├── 处理读请求
│   ├── 参与投票
│   └── 同步 Leader 状态
└── Observer (观察者)
    ├── 处理读请求
    └── 不参与投票

ZNode (数据节点)
├── 持久节点
├── 临时节点
├── 顺序节点
└── 容器节点

Watcher (监听器)
├── 节点创建
├── 节点删除
├── 数据变化
└── 子节点变化
*/
```

## 3. ZooKeeper 数据模型

### 3.1 ZNode 类型

- **持久节点**：节点创建后一直存在，直到显式删除
- **临时节点**：节点绑定会话，会话结束自动删除
- **顺序节点**：节点名称自动添加序号
- **容器节点**：用于分组，子节点全部删除后自动删除

```java
// ZNode 类型示例
/*
持久节点: /config/database
临时节点: /locks/lock_001
顺序节点: /tasks/task_0000000001
临时顺序节点: /election/node_0000000001
*/
```

### 3.2 ZNode 结构

每个 ZNode 包含以下信息：
- **数据**：存储的数据（最大 1MB）
- **版本**：数据版本号
- **ACL**：访问控制列表
- **子节点**：子节点列表
- **状态**：节点状态信息

## 4. ZooKeeper 操作

### 4.1 Java API 操作

```java
import org.apache.zookeeper.*;
import org.apache.zookeeper.data.Stat;

import java.io.IOException;
import java.util.List;
import java.util.concurrent.CountDownLatch;

public class ZooKeeperOperations implements Watcher {
    private ZooKeeper zk;
    private CountDownLatch connectedLatch = new CountDownLatch(1);
    
    public ZooKeeperOperations(String connectString) throws IOException, InterruptedException {
        zk = new ZooKeeper(connectString, 30000, this);
        connectedLatch.await();
    }
    
    @Override
    public void process(WatchedEvent event) {
        if (event.getState() == Event.KeeperState.SyncConnected) {
            connectedLatch.countDown();
        }
        
        System.out.println("事件类型: " + event.getType());
        System.out.println("事件路径: " + event.getPath());
        System.out.println("事件状态: " + event.getState());
    }
    
    // 创建节点
    public void createNode(String path, String data, CreateMode mode) throws Exception {
        byte[] dataBytes = data != null ? data.getBytes() : null;
        String createdPath = zk.create(path, dataBytes, 
                                      ZooDefs.Ids.OPEN_ACL_UNSAFE, mode);
        System.out.println("节点创建成功: " + createdPath);
    }
    
    // 获取节点数据
    public String getNodeData(String path) throws Exception {
        Stat stat = new Stat();
        byte[] data = zk.getData(path, false, stat);
        System.out.println("数据版本: " + stat.getVersion());
        System.out.println("数据长度: " + stat.getDataLength());
        return new String(data);
    }
    
    // 更新节点数据
    public void updateNodeData(String path, String data) throws Exception {
        Stat stat = zk.exists(path, false);
        if (stat != null) {
            zk.setData(path, data.getBytes(), stat.getVersion());
            System.out.println("节点数据更新成功");
        } else {
            System.out.println("节点不存在");
        }
    }
    
    // 删除节点
    public void deleteNode(String path) throws Exception {
        Stat stat = zk.exists(path, false);
        if (stat != null) {
            zk.delete(path, stat.getVersion());
            System.out.println("节点删除成功");
        } else {
            System.out.println("节点不存在");
        }
    }
    
    // 检查节点是否存在
    public boolean nodeExists(String path) throws Exception {
        Stat stat = zk.exists(path, false);
        return stat != null;
    }
    
    // 获取子节点
    public List<String> getChildren(String path) throws Exception {
        List<String> children = zk.getChildren(path, false);
        System.out.println("子节点数量: " + children.size());
        return children;
    }
    
    // 设置监听器
    public void watchNode(String path) throws Exception {
        zk.getData(path, true, null);
        System.out.println("已设置监听器: " + path);
    }
    
    // 关闭连接
    public void close() throws InterruptedException {
        zk.close();
    }
    
    public static void main(String[] args) throws Exception {
        ZooKeeperOperations zkOps = new ZooKeeperOperations("localhost:2181");
        
        // 创建持久节点
        zkOps.createNode("/config", "database config", CreateMode.PERSISTENT);
        zkOps.createNode("/config/database", "mysql", CreateMode.PERSISTENT);
        
        // 创建临时节点
        zkOps.createNode("/locks/lock_001", "locked", CreateMode.EPHEMERAL);
        
        // 创建顺序节点
        zkOps.createNode("/tasks/task-", "task data", CreateMode.PERSISTENT_SEQUENTIAL);
        
        // 获取节点数据
        String data = zkOps.getNodeData("/config/database");
        System.out.println("节点数据: " + data);
        
        // 更新节点数据
        zkOps.updateNodeData("/config/database", "postgresql");
        
        // 获取子节点
        List<String> children = zkOps.getChildren("/config");
        System.out.println("子节点: " + children);
        
        // 设置监听器
        zkOps.watchNode("/config/database");
        
        // 删除节点
        zkOps.deleteNode("/locks/lock_001");
        
        zkOps.close();
    }
}
```

```java
    zk.create(path, dataBytes, ZooDefs.Ids.OPEN_ACL_UNSAFE, mode,
```

## 5. ZooKeeper 应用场景

### 5.1 分布式锁

```java
import org.apache.zookeeper.*;
import org.apache.zookeeper.data.Stat;

import java.io.IOException;
import java.util.Collections;
import java.util.List;
import java.util.concurrent.CountDownLatch;

public class DistributedLock implements Watcher {
    private ZooKeeper zk;
    private String lockPath;
    private String currentLock;
    private CountDownLatch connectedLatch = new CountDownLatch(1);
    private CountDownLatch acquiredLatch = new CountDownLatch(1);
    
    public DistributedLock(String connectString, String lockPath) throws IOException, InterruptedException {
        this.lockPath = lockPath;
        zk = new ZooKeeper(connectString, 30000, this);
        connectedLatch.await();
        
        // 确保锁路径存在
        try {
            if (zk.exists(lockPath, false) == null) {
                zk.create(lockPath, null, ZooDefs.Ids.OPEN_ACL_UNSAFE, 
                         CreateMode.PERSISTENT);
            }
        } catch (KeeperException | InterruptedException e) {
            e.printStackTrace();
        }
    }
    
    @Override
    public void process(WatchedEvent event) {
        if (event.getState() == Event.KeeperState.SyncConnected) {
            connectedLatch.countDown();
        }
        
        if (event.getType() == Event.EventType.NodeDeleted) {
            acquiredLatch.countDown();
        }
    }
    
    // 获取锁
    public boolean lock() throws Exception {
        // 创建临时顺序节点
        currentLock = zk.create(lockPath + "/lock-", null, 
                               ZooDefs.Ids.OPEN_ACL_UNSAFE, 
                               CreateMode.EPHEMERAL_SEQUENTIAL);
        
        System.out.println("创建锁节点: " + currentLock);
        
        // 获取所有锁节点
        while (true) {
            List<String> children = zk.getChildren(lockPath, false);
            Collections.sort(children);
            
            String currentNode = currentLock.substring(lockPath.length() + 1);
            int currentIndex = children.indexOf(currentNode);
            
            if (currentIndex == 0) {
                // 获取到锁
                System.out.println("获取锁成功");
                return true;
            } else {
                // 监听前一个节点
                String previousNode = children.get(currentIndex - 1);
                String previousPath = lockPath + "/" + previousNode;
                
                Stat stat = zk.exists(previousPath, true);
                if (stat == null) {
                    // 前一个节点已删除，重新尝试
                    continue;
                }
                
                // 等待前一个节点删除
                acquiredLatch = new CountDownLatch(1);
                acquiredLatch.await();
            }
        }
    }
    
    // 释放锁
    public void unlock() throws Exception {
        if (currentLock != null) {
            zk.delete(currentLock, -1);
            System.out.println("释放锁: " + currentLock);
            currentLock = null;
        }
    }
    
    public void close() throws InterruptedException {
        zk.close();
    }
    
    public static void main(String[] args) throws Exception {
        DistributedLock lock = new DistributedLock("localhost:2181", "/locks");
        
        try {
            // 获取锁
            if (lock.lock()) {
                // 执行业务逻辑
                System.out.println("执行业务逻辑...");
                Thread.sleep(5000);
            }
        } finally {
            // 释放锁
            lock.unlock();
            lock.close();
        }
    }
}
```

### 5.2 配置中心

```java
import org.apache.zookeeper.*;
import org.apache.zookeeper.data.Stat;

import java.io.IOException;
import java.util.concurrent.CountDownLatch;

public class ConfigCenter implements Watcher {
    private ZooKeeper zk;
    private String configPath;
    private CountDownLatch connectedLatch = new CountDownLatch(1);
    private ConfigChangeListener listener;
    
    public interface ConfigChangeListener {
        void onConfigChanged(String key, String value);
    }
    
    public ConfigCenter(String connectString, String configPath, 
                       ConfigChangeListener listener) throws IOException, InterruptedException {
        this.configPath = configPath;
        this.listener = listener;
        zk = new ZooKeeper(connectString, 30000, this);
        connectedLatch.await();
    }
    
    @Override
    public void process(WatchedEvent event) {
        if (event.getState() == Event.KeeperState.SyncConnected) {
            connectedLatch.countDown();
        }
        
        if (event.getType() == Event.EventType.NodeDataChanged) {
            try {
                String value = getConfig(event.getPath());
                String key = event.getPath().substring(configPath.length() + 1);
                listener.onConfigChanged(key, value);
                
                // 重新设置监听
                zk.getData(event.getPath(), true, null);
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
    }
    
    // 设置配置
    public void setConfig(String key, String value) throws Exception {
        String path = configPath + "/" + key;
        Stat stat = zk.exists(path, false);
        
        if (stat == null) {
            zk.create(path, value.getBytes(), 
                     ZooDefs.Ids.OPEN_ACL_UNSAFE, 
                     CreateMode.PERSISTENT);
        } else {
            zk.setData(path, value.getBytes(), stat.getVersion());
        }
        
        System.out.println("配置设置成功: " + key + " = " + value);
    }
    
    // 获取配置
    public String getConfig(String key) throws Exception {
        String path = configPath + "/" + key;
        Stat stat = new Stat();
        byte[] data = zk.getData(path, true, stat);
        return new String(data);
    }
    
    // 监听配置变化
    public void watchConfig(String key) throws Exception {
        String path = configPath + "/" + key;
        zk.getData(path, true, null);
        System.out.println("开始监听配置: " + key);
    }
    
    // 删除配置
    public void deleteConfig(String key) throws Exception {
        String path = configPath + "/" + key;
        Stat stat = zk.exists(path, false);
        if (stat != null) {
            zk.delete(path, stat.getVersion());
            System.out.println("配置删除成功: " + key);
        }
    }
    
    public void close() throws InterruptedException {
        zk.close();
    }
    
    public static void main(String[] args) throws Exception {
        ConfigCenter configCenter = new ConfigCenter("localhost:2181", "/config",
            new ConfigChangeListener() {
                @Override
                public void onConfigChanged(String key, String value) {
                    System.out.println("配置变化: " + key + " = " + value);
                }
            });
        
        // 设置配置
        configCenter.setConfig("database.url", "jdbc:mysql://localhost:3306/mydb");
        configCenter.setConfig("database.user", "root");
        configCenter.setConfig("database.password", "password");
        
        // 获取配置
        String url = configCenter.getConfig("database.url");
        System.out.println("配置值: " + url);
        
        // 监听配置变化
        configCenter.watchConfig("database.url");
        
        // 模拟配置变化
        Thread.sleep(1000);
        configCenter.setConfig("database.url", "jdbc:mysql://localhost:3306/newdb");
        
        Thread.sleep(2000);
        configCenter.close();
    }
}
```

### 5.3 服务发现

```java
import org.apache.zookeeper.*;
import org.apache.zookeeper.data.Stat;

import java.io.IOException;
import java.util.ArrayList;
import java.util.List;
import java.util.concurrent.CountDownLatch;

public class ServiceDiscovery implements Watcher {
    private ZooKeeper zk;
    private String servicePath;
    private CountDownLatch connectedLatch = new CountDownLatch(1);
    private List<String> services = new ArrayList<>();
    
    public ServiceDiscovery(String connectString, String servicePath) 
            throws IOException, InterruptedException {
        this.servicePath = servicePath;
        zk = new ZooKeeper(connectString, 30000, this);
        connectedLatch.await();
        
        // 确保服务路径存在
        try {
            if (zk.exists(servicePath, false) == null) {
                zk.create(servicePath, null, ZooDefs.Ids.OPEN_ACL_UNSAFE, 
                         CreateMode.PERSISTENT);
            }
        } catch (KeeperException | InterruptedException e) {
            e.printStackTrace();
        }
        
        // 初始化服务列表
        updateServices();
    }
    
    @Override
    public void process(WatchedEvent event) {
        if (event.getState() == Event.KeeperState.SyncConnected) {
            connectedLatch.countDown();
        }
        
        if (event.getType() == Event.EventType.NodeChildrenChanged) {
            try {
                updateServices();
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
    }
    
    // 注册服务
    public void registerService(String serviceName, String address) throws Exception {
        String path = servicePath + "/" + serviceName;
        Stat stat = zk.exists(path, false);
        
        if (stat == null) {
            zk.create(path, null, ZooDefs.Ids.OPEN_ACL_UNSAFE, 
                     CreateMode.PERSISTENT);
        }
        
        // 创建临时节点
        String serviceNode = zk.create(path + "/service-", 
                                      address.getBytes(),
                                      ZooDefs.Ids.OPEN_ACL_UNSAFE, 
                                      CreateMode.EPHEMERAL_SEQUENTIAL);
        
        System.out.println("服务注册成功: " + serviceNode);
    }
    
    // 发现服务
    public List<String> discoverServices(String serviceName) throws Exception {
        String path = servicePath + "/" + serviceName;
        List<String> children = zk.getChildren(path, true);
        
        List<String> addresses = new ArrayList<>();
        for (String child : children) {
            String childPath = path + "/" + child;
            byte[] data = zk.getData(childPath, false, null);
            addresses.add(new String(data));
        }
        
        return addresses;
    }
    
    // 更新服务列表
    private void updateServices() throws Exception {
        List<String> children = zk.getChildren(servicePath, true);
        services.clear();
        services.addAll(children);
        System.out.println("服务列表更新: " + services);
    }
    
    // 获取所有服务
    public List<String> getAllServices() {
        return new ArrayList<>(services);
    }
    
    public void close() throws InterruptedException {
        zk.close();
    }
    
    public static void main(String[] args) throws Exception {
        ServiceDiscovery discovery = new ServiceDiscovery("localhost:2181", "/services");
        
        // 注册服务
        discovery.registerService("user-service", "192.168.1.100:8080");
        discovery.registerService("user-service", "192.168.1.101:8080");
        discovery.registerService("order-service", "192.168.1.102:8080");
        
        // 发现服务
        List<String> userServices = discovery.discoverServices("user-service");
        System.out.println("用户服务: " + userServices);
        
        List<String> orderServices = discovery.discoverServices("order-service");
        System.out.println("订单服务: " + orderServices);
        
        // 获取所有服务
        List<String> allServices = discovery.getAllServices();
        System.out.println("所有服务: " + allServices);
        
        Thread.sleep(5000);
        discovery.close();
    }
}
```

## 6. ZooKeeper 面试题

### 6.1 基础概念

**Q1: ZooKeeper 的作用是什么？**

ZooKeeper 是一个分布式协调服务，用于：
- 配置管理
- 命名服务
- 分布式锁
- 集群管理
- 服务发现

**Q2: ZooKeeper 的数据模型有什么特点？**

- 树形结构，类似文件系统
- 节点称为 ZNode
- 支持多种节点类型（持久、临时、顺序）
- 每个节点可以存储数据（最大 1MB）
- 支持版本控制和监听机制

### 6.2 技术原理

**Q3: ZooKeeper 的工作原理是怎样的？**

ZooKeeper 采用 ZAB 协议：
- **Leader 选举**：集群启动时选举 Leader
- **数据同步**：Leader 与 Follower 同步数据
- **写请求**：所有写请求通过 Leader 处理
- **读请求**：可以从任意节点读取
- **广播更新**：Leader 将更新广播给所有 Follower

**Q4: ZooKeeper 如何保证数据一致性？**

- 使用 ZAB 协议保证一致性
- 通过 Leader 选举保证唯一性
- 使用版本号和事务 ID 保证顺序性
- 通过多数派机制保证数据可靠性

### 6.3 应用场景

**Q5: ZooKeeper 的典型应用场景有哪些？**

- **分布式锁**：实现互斥访问
- **配置中心**：集中管理配置
- **服务发现**：动态服务注册和发现
- **集群管理**：监控集群状态
- **Leader 选举**：选举集群 Leader

**Q6: 如何实现分布式锁？**

1. 创建临时顺序节点
2. 获取所有锁节点并排序
3. 判断自己是否是最小的节点
4. 如果不是，监听前一个节点
5. 前一个节点删除时获取锁
6. 使用完成后删除自己的节点

## 7. 总结

ZooKeeper 作为分布式系统的重要协调服务，具有以下特点：

1. **简单易用**：提供简单的 API，易于集成
2. **高可用性**：集群部署，避免单点故障
3. **高性能**：针对读操作优化，支持高并发
4. **一致性**：保证数据的一致性和顺序性

掌握 ZooKeeper 的使用方法和应用场景，对于构建可靠的分布式系统具有重要意义。通过合理使用 ZooKeeper，可以简化分布式系统的开发，提高系统的可靠性和可维护性。
