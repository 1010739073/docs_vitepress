# HBase NoSQL 数据库详解

## 1. HBase 概述

HBase 是建立在 HDFS 之上的分布式、面向列的 NoSQL 数据库，提供高可靠性、高性能、列式可伸缩存储，用于处理海量结构化数据。

### 1.1 HBase 特性

- **分布式存储**：数据自动分片和负载均衡
- **高可靠性**：数据自动复制，故障自动恢复
- **高性能**：支持随机读写，毫秒级响应
- **列式存储**：灵活的列族和列设计
- **可扩展性**：支持线性扩展，处理 PB 级数据

## 2. HBase 架构

### 2.1 核心组件

- **HMaster**：主节点，负责表管理和负载均衡
- **HRegionServer**：从节点，负责数据读写
- **HRegion**：数据分片，表的水平分割
- **Store**：列族存储，每个列族对应一个 Store
- **MemStore**：内存缓存，写入缓冲
- **HFile**：磁盘存储，数据文件

### 2.2 架构原理

```java
// HBase 架构示意
/*
HMaster (主节点)
├── 表管理
├── Region 分配
├── 负载均衡
└── 故障恢复

HRegionServer (从节点)
├── HRegion (数据分片)

└── RegionServer 服务

ZooKeeper (协调服务)
├── Master 选举
├── RegionServer 注册
└── 元数据管理
*/
```

## 3. HBase 数据模型

### 3.1 数据模型概念

- **Table**：表，数据存储的基本单位
- **RowKey**：行键，表的唯一标识
- **Column Family**：列族，列的集合
- **Column Qualifier**：列限定符，具体的列
- **Timestamp**：时间戳，版本控制
- **Cell**：单元格，存储具体数据

```java
// HBase 数据模型示意
/*
Table: users
RowKey: user_001
Column Family: info
  - name: "张三"
  - age: "25"
  - email: "zhangsan@example.com"
Column Family: address
  - city: "北京"
  - street: "朝阳区"
  - zip: "100000"
Timestamp: 1234567890
*/
```

### 3.2 数据模型特点

- **稀疏性**：空值不占用存储空间
- **多版本**：每个单元格可存储多个版本
- **排序**：数据按 RowKey 字典序排序
- **列族**：列族需要预先定义，列可以动态添加

## 4. HBase 操作

### 4.1 Java API 操作

```java
import org.apache.hadoop.hbase.*;
import org.apache.hadoop.hbase.client.*;
import org.apache.hadoop.hbase.util.Bytes;

public class HBaseOperations {
    private Connection connection;
    private Admin admin;
    
    public HBaseOperations() throws Exception {
        Configuration config = HBaseConfiguration.create();
        config.set("hbase.zookeeper.quorum", "localhost");
        config.set("hbase.zookeeper.property.clientPort", "2181");
        
        connection = ConnectionFactory.createConnection(config);
        admin = connection.getAdmin();
    }
    
    // 创建表
    public void createTable(String tableName, String[] columnFamilies) throws Exception {
        if (admin.tableExists(TableName.valueOf(tableName))) {
            System.out.println("表已存在: " + tableName);
            return;
        }
        
        TableDescriptorBuilder tableBuilder = TableDescriptorBuilder.newBuilder(
            TableName.valueOf(tableName));
        
        for (String cf : columnFamilies) {
            ColumnFamilyDescriptorBuilder cfBuilder = 
                ColumnFamilyDescriptorBuilder.newBuilder(Bytes.toBytes(cf));
            tableBuilder.setColumnFamily(cfBuilder.build());
        }
        
        admin.createTable(tableBuilder.build());
        System.out.println("表创建成功: " + tableName);
    }
    
    // 插入数据
    public void putData(String tableName, String rowKey, 
                       String columnFamily, String column, String value) throws Exception {
        Table table = connection.getTable(TableName.valueOf(tableName));
        
        Put put = new Put(Bytes.toBytes(rowKey));
        put.addColumn(Bytes.toBytes(columnFamily), 
                     Bytes.toBytes(column), 
                     Bytes.toBytes(value));
        
        table.put(put);
        table.close();
        System.out.println("数据插入成功");
    }
    
    // 获取数据
    public void getData(String tableName, String rowKey) throws Exception {
        Table table = connection.getTable(TableName.valueOf(tableName));
        
        Get get = new Get(Bytes.toBytes(rowKey));
        Result result = table.get(get);
        
        for (Cell cell : result.rawCells()) {
            String cf = Bytes.toString(CellUtil.cloneFamily(cell));
            String column = Bytes.toString(CellUtil.cloneQualifier(cell));
            String value = Bytes.toString(CellUtil.cloneValue(cell));
            long timestamp = cell.getTimestamp();
            
            System.out.println("列族: " + cf + 
                             ", 列: " + column + 
                             ", 值: " + value + 
                             ", 时间戳: " + timestamp);
        }
        
        table.close();
    }
    
    // 扫描数据
    public void scanData(String tableName) throws Exception {
        Table table = connection.getTable(TableName.valueOf(tableName));
        
        Scan scan = new Scan();
        ResultScanner scanner = table.getScanner(scan);
        
        for (Result result : scanner) {
            String rowKey = Bytes.toString(result.getRow());
            System.out.println("RowKey: " + rowKey);
            
            for (Cell cell : result.rawCells()) {
                String cf = Bytes.toString(CellUtil.cloneFamily(cell));
                String column = Bytes.toString(CellUtil.cloneQualifier(cell));
                String value = Bytes.toString(CellUtil.cloneValue(cell));
                
                System.out.println("  " + cf + ":" + column + " = " + value);
            }
        }
        
        scanner.close();
        table.close();
    }
    
    // 删除数据
    public void deleteData(String tableName, String rowKey) throws Exception {
        Table table = connection.getTable(TableName.valueOf(tableName));
        
        Delete delete = new Delete(Bytes.toBytes(rowKey));
        table.delete(delete);
        table.close();
        System.out.println("数据删除成功");
    }
    
    // 删除表
    public void deleteTable(String tableName) throws Exception {
        if (!admin.tableExists(TableName.valueOf(tableName))) {
            System.out.println("表不存在: " + tableName);
            return;
        }
        
        admin.disableTable(TableName.valueOf(tableName));
        admin.deleteTable(TableName.valueOf(tableName));
        System.out.println("表删除成功: " + tableName);
    }
    
    public void close() throws Exception {
        admin.close();
        connection.close();
    }
    
    public static void main(String[] args) throws Exception {
        HBaseOperations hbase = new HBaseOperations();
        
        // 创建表
        String[] columnFamilies = {"info", "address"};
        hbase.createTable("users", columnFamilies);
        
        // 插入数据
        hbase.putData("users", "user_001", "info", "name", "张三");
        hbase.putData("users", "user_001", "info", "age", "25");
        hbase.putData("users", "user_001", "address", "city", "北京");
        
        // 获取数据
        hbase.getData("users", "user_001");
        
        // 扫描数据
        hbase.scanData("users");
        
        // 删除数据
        hbase.deleteData("users", "user_001");
        
        // 删除表
        hbase.deleteTable("users");
        
        hbase.close();
    }
}
```

## 6. HBase 优化

### 6.1 表设计优化

```java
// RowKey 设计原则
// 1. 长度原则：RowKey 越短越好，建议 16 字节以内
// 2. 散列原则：使用散列算法避免热点问题
// 3. 唯一性：确保 RowKey 唯一

// 好的 RowKey 设计示例
public String designRowKey(String userId, long timestamp) {
    // 使用 MD5 散列避免热点
    String md5Hash = DigestUtils.md5Hex(userId);
    String hashPrefix = md5Hash.substring(0, 4);
    
    // 组合 RowKey
    return hashPrefix + "_" + userId + "_" + timestamp;
}

// 反转时间戳实现最新数据在前
public String designRowKeyWithReverseTimestamp(String userId) {
    long reverseTimestamp = Long.MAX_VALUE - System.currentTimeMillis();
    return userId + "_" + reverseTimestamp;
}
```

### 6.2 列族设计

```java
// 列族设计原则
// 1. 列族数量不宜过多，建议 1-3 个
// 2. 将经常一起查询的列放在同一列族
// 3. 合理设置列族属性

// 创建优化的列族
public void createOptimizedTable(String tableName) throws Exception {
    TableDescriptorBuilder tableBuilder = TableDescriptorBuilder.newBuilder(
        TableName.valueOf(tableName));
    
    // 列族1：经常查询的数据
    ColumnFamilyDescriptorBuilder cf1Builder = 
        ColumnFamilyDescriptorBuilder.newBuilder(Bytes.toBytes("info"));
    cf1Builder.setMaxVersions(1);  // 只保留一个版本
    cf1Builder.setCompressionType(Compression.Algorithm.SNAPPY);  // 启用压缩
    cf1Builder.setBloomFilterType(BloomType.ROW);  // 启用布隆过滤器
    cf1Builder.setBlockCacheEnabled(true);  // 启用块缓存
    tableBuilder.setColumnFamily(cf1Builder.build());
    
    // 列族2：不常查询的大数据
    ColumnFamilyDescriptorBuilder cf2Builder = 
        ColumnFamilyDescriptorBuilder.newBuilder(Bytes.toBytes("data"));
    cf2Builder.setMaxVersions(3);  // 保留多个版本
    cf2Builder.setCompressionType(Compression.Algorithm.GZIP);  // 使用 GZIP 压缩
    cf2Builder.setBloomFilterType(BloomType.ROWCOL);  // 启用行列布隆过滤器
    cf2Builder.setBlockCacheEnabled(false);  // 禁用块缓存
    tableBuilder.setColumnFamily(cf2Builder.build());
    
    admin.createTable(tableBuilder.build());
}
```

## 7. HBase 面试题

### 7.1 基础概念

**Q1: HBase 的特点是什么？**

- 分布式、可扩展的 NoSQL 数据库
- 列式存储，支持动态列
- 高可靠性、高可用性
- 支持随机读写，毫秒级响应
- 基于 HDFS，支持海量数据存储

**Q2: HBase 的数据模型有什么特点？**

- 表由行和列组成
- 行键是唯一的，按字典序排序
- 列族需要预先定义，列可以动态添加
- 每个单元格可以存储多个版本
- 数据按列族存储，支持稀疏性

### 7.2 技术原理

**Q3: HBase 的读写流程是怎样的？**

**写流程**：
1. 客户端向 HRegionServer 发送写请求
2. 数据写入 WAL (Write-Ahead Log)
3. 数据写入 MemStore
4. MemStore 满后 flush 到 HFile

**读流程**：
1. 客户端向 HRegionServer 发送读请求
2. 检查 BlockCache
3. 检查 MemStore
4. 读取 HFile
5. 合并结果返回

**Q4: HBase 如何实现数据一致性？**

- 使用 WAL 保证数据持久性
- 通过 MVCC 实现读写一致性
- 使用 ZooKeeper 协调集群状态
- RegionServer 故障时从 WAL 恢复数据

### 7.3 优化问题

**Q5: 如何优化 HBase 性能？**

- 合理设计 RowKey，避免热点问题
- 优化列族设计，控制列族数量
- 启用压缩和布隆过滤器
- 调整 MemStore 和 BlockCache 大小
- 使用批量操作减少 RPC 调用
- 合理设置预分区

**Q6: HBase 的 RowKey 设计有什么原则？**

- 唯一性：确保 RowKey 唯一
- 长度：尽量短，建议 16 字节以内
- 散列性：避免热点，使用散列算法
- 排序：利用字典序优化查询
- 固定长度：便于查询和分区

### 8.1 完整的用户管理系统

```java
import org.apache.hadoop.hbase.*;
import org.apache.hadoop.hbase.client.*;
import org.apache.hadoop.hbase.util.Bytes;
import org.apache.commons.codec.digest.DigestUtils;

public class UserManagementSystem {
    private Connection connection;
    private Admin admin;
    
    public UserManagementSystem() throws Exception {
        Configuration config = HBaseConfiguration.create();
        config.set("hbase.zookeeper.quorum", "localhost");
        config.set("hbase.zookeeper.property.clientPort", "2181");
        
        connection = ConnectionFactory.createConnection(config);
        admin = connection.getAdmin();
    }
    
    // 创建用户表
    public void createUserTable() throws Exception {
        TableName tableName = TableName.valueOf("users");
        
        if (admin.tableExists(tableName)) {
            System.out.println("用户表已存在");
            return;
        }
        
        TableDescriptorBuilder tableBuilder = TableDescriptorBuilder.newBuilder(tableName);
        
        // 基本信息列族
        ColumnFamilyDescriptorBuilder infoCF = 
            ColumnFamilyDescriptorBuilder.newBuilder(Bytes.toBytes("info"));
        infoCF.setMaxVersions(1);
        infoCF.setCompressionType(Compression.Algorithm.SNAPPY);
        infoCF.setBloomFilterType(BloomType.ROW);
        tableBuilder.setColumnFamily(infoCF.build());
        
        // 扩展信息列族
        ColumnFamilyDescriptorBuilder extCF = 
            ColumnFamilyDescriptorBuilder.newBuilder(Bytes.toBytes("ext"));
        extCF.setMaxVersions(3);
        extCF.setCompressionType(Compression.Algorithm.GZIP);
        tableBuilder.setColumnFamily(extCF.build());
        
        // 创建预分区
        byte[][] splitKeys = new byte[10][];
        for (int i = 0; i < 10; i++) {
            splitKeys[i] = Bytes.toBytes(String.format("%04d", i * 1000));
        }
        
        admin.createTable(tableBuilder.build(), splitKeys);
        System.out.println("用户表创建成功");
    }
    
    // 生成 RowKey
    private String generateRowKey(String userId) {
        String md5Hash = DigestUtils.md5Hex(userId);
        String hashPrefix = md5Hash.substring(0, 4);
        return hashPrefix + "_" + userId;
    }
    
    // 添加用户
    public void addUser(User user) throws Exception {
        Table table = connection.getTable(TableName.valueOf("users"));
        
        String rowKey = generateRowKey.getUserId());
        Put put = new Put(Bytes.toBytes(rowKey));
        
        // 基本信息
        put.addColumn(Bytes.toBytes("info"), Bytes.toBytes("name"), 
                     Bytes.toBytes(user.getName()));
        put.addColumn(Bytes.toBytes("info"), Bytes.toBytes("email"), 
                     Bytes.toBytes(user.getEmail()));
        put.addColumn(Bytes.toBytes("info"), Bytes.toBytes("phone"), 
                     Bytes.toBytes(user.getPhone()));
        put.addColumn(Bytes.toBytes("info"), Bytes.toBytes("created_time"), 
                     Bytes.toBytes(System.currentTimeMillis()));
        
        // 扩展信息
        if (user.getAddress() != null) {
            put.addColumn(Bytes.toBytes("ext"), Bytes.toBytes("address"), 
                         Bytes.toBytes(user.getAddress()));
        }
        if (user.getCompany() != null) {
            put.addColumn(Bytes.toBytes("ext"), Bytes.toBytes("company"), 
                         Bytes.toBytes(user.getCompany()));
        }
        
        table.put(put);
        table.close();
        System.out.println("用户添加成功: " + user.getUserId());
    }
    
    // 获取用户信息
    public User getUser(String userId) throws Exception {
        Table table = connection.getTable(TableName.valueOf("users"));
        
        String rowKey = generateRowKey(userId);
        Get get = new Get(Bytes.toBytes(rowKey));
        Result result = table.get(get);
        
        if (result.isEmpty()) {
            table.close();
            return null;
        }
        
        User user = new User();
        user.setUserId(userId);
        
        // 读取基本信息
        byte[] name = result.getValue(Bytes.toBytes("info"), Bytes.toBytes("name"));
        if (name != null) {
            user.setName(Bytes.toString(name));
        }
        
        byte[] email = result.getValue(Bytes.toBytes("info"), Bytes.toBytes("email"));
        if (email != null) {
            user.setEmail(Bytes.toString(email));
        }
        
        byte[] phone = result.getValue(Bytes.toBytes("info"), Bytes.toBytes("phone"));
        if (phone != null) {
            user.setPhone(Bytes.toString(phone));
        }
        
        // 读取扩展信息
        byte[] address = result.getValue(Bytes.toBytes("ext"), Bytes.toBytes("address"));
        if (address != null) {
            user.setAddress(Bytes.toString(address));
        }
        
        byte[] company = result.getValue(Bytes.toBytes("ext"), Bytes.toBytes("company"));
        if (company != null) {
            user.setCompany(Bytes.toString(company));
        }
        
        table.close();
        return user;
    }
    
    // 更新用户信息
    public void updateUser(User user) throws Exception {
        Table table = connection.getTable(TableName.valueOf("users"));
        
        String rowKey = generateRowKey(user.getUserId());
        Put put = new Put(Bytes.toBytes(rowKey));
        
        if (user.getName() != null) {
            put.addColumn(Bytes.toBytes("info"), Bytes.toBytes("name"), 
                         Bytes.toBytes(user.getName()));
        }
        if (user.getEmail() != null) {
            put.addColumn(Bytes.toBytes("info"), Bytes.toBytes("email"), 
                         Bytes.toBytes(user.getEmail()));
        }
        if (user.getPhone() != null) {
            put.addColumn(Bytes.toBytes("info"), Bytes.toBytes("phone"), 
                         Bytes.toBytes(user.getPhone()));
        }
        
        table.put(put);
        table.close();
        System.out.println("用户信息更新成功: " + user.getUserId());
    }
    
    // 删除用户
    public void deleteUser(String userId) throws Exception {
        Table table = connection.getTable(TableName.valueOf("users"));
        
        String rowKey = generateRowKey(userId);
        Delete delete = new Delete(Bytes.toBytes(rowKey));
        table.delete(delete);
        table.close();
        System.out.println("用户删除成功: " + userId);
    }
    
    // 搜索用户
    public List<User> searchUsers(String prefix, int limit) throws Exception {
        Table table = connection.getTable(TableName.valueOf("users"));
        
        Scan scan = new Scan();
        scan.setRowPrefixFilter(Bytes.toBytes(prefix));
        scan.setLimit(limit);
        
        ResultScanner scanner = table.getScanner(scan);
        List<User> users = new ArrayList<>();
        
        for (Result result : scanner) {
            String rowKey = Bytes.toString(result.getRow());
            String userId = rowKey.substring(5);  // 去掉 hash 前缀
            
            User user = new User();
            user.setUserId(userId);
            
            byte[] name = result.getValue(Bytes.toBytes("info"), Bytes.toBytes("name"));
            if (name != null) {
                user.setName(Bytes.toString(name));
            }
            
            byte[] email = result.getValue(Bytes.toBytes("info"), Bytes.toBytes("email"));
            if (email != null) {
                user.setEmail(Bytes.toString(email));
            }
            
            users.add(user);
        }
        
        scanner.close();
        table.close();
        return users;
    }
```

## 9. 总结

HBase 作为大数据领域的重要 NoSQL 数据库，具有以下特点：

1. **高性能**：支持随机读写，毫秒级响应
2. **高可靠性**：数据自动复制，故障自动恢复
3. **可扩展性**：支持线性扩展，处理 PB 级数据
4. **灵活性**：列式存储，支持动态列

掌握 HBase 的设计原理和使用方法，对于构建高性能的大数据存储系统具有重要意义。通过合理的表设计、RowKey 设计和性能调优，可以充分发挥 HBase 的优势，满足各种大数据存储需求。
