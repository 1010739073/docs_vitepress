# 3. 安装与配置

## 环境要求

### 系统要求

| 要求 | 说明 |
|------|------|
| Java 版本 | 8u201+, 11 (推荐) |
| 操作系统 | Linux, macOS, Windows |
| 内存 | 至少 2GB |
| 文件描述符 | 至少 65535 |

### 内核参数设置

```bash
# 修改文件描述符限制
echo "elasticsearch soft nofile 65535" >> /etc/security/limits.conf
echo "elasticsearch hard nofile 65535" >> /etc/security/limits.conf

# 修改线程数限制
echo "elasticsearch soft nproc 4096" >> /etc/security/limits.conf
echo "elasticsearch hard nproc 4096" >> /etc/security/limits.conf

# 禁用内存交换
echo "vm.swappiness=1" >> /etc/sysctl.conf
sysctl -p
```

## 安装方式

### Docker 安装

```bash
# 拉取镜像
docker pull elasticsearch:8.11.0

# 运行单节点
docker run -d \
  --name elasticsearch \
  -p 9200:9200 \
  -p 9300:9300 \
  -e "discovery.type=single-node" \
  -e "ES_JAVA_OPTS=-Xms512m -Xmx512m" \
  elasticsearch:8.11.0
```

### 二进制安装

```bash
# 下载
wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-8.11.0-linux-x86_64.tar.gz

# 解压
tar -xzf elasticsearch-8.11.0-linux-x86_64.tar.gz
cd elasticsearch-8.11.0

# 创建用户
useradd elasticsearch
chown -R elasticsearch:elasticsearch .

# 启动
su - elasticsearch
./bin/elasticsearch
```

## 配置文件详解

### elasticsearch.yml

```yaml
# 集群名称
cluster.name: my-es-cluster

# 节点名称
node.name: node-1

# 节点角色
node.roles: [master, data, ingest]

# 数据目录
path.data: /var/lib/elasticsearch

# 日志目录
path.logs: /var/log/elasticsearch

# 网络设置
network.host: 0.0.0.0
http.port: 9200

# 发现设置
discovery.seed_hosts: ["node-1", "node-2", "node-3"]
cluster.initial_master_nodes: ["node-1", "node-2", "node-3"]

# 内存锁定
bootstrap.memory_lock: true

# 线程池设置
thread_pool.search.size: 10
thread_pool.search.queue_size: 1000
```

### 配置项说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `cluster.name` | 集群名称 | elasticsearch |
| `node.name` | 节点名称 | 自动生成 |
| `node.master` | 是否可成为主节点 | true |
| `node.data` | 是否存储数据 | true |
| `network.host` | 绑定地址 | 127.0.0.1 |
| `http.port` | HTTP端口 | 9200 |
| `discovery.seed_hosts` | 种子节点列表 | [] |
| `cluster.initial_master_nodes` | 初始主节点 | [] |

## 安全配置

### HTTPS 配置

```yaml
# 启用 HTTPS
xpack.security.enabled: true
xpack.security.transport.ssl.enabled: true
xpack.security.transport.ssl.verification_mode: certificate
xpack.security.transport.ssl.keystore.path: certs/elastic-certificates.p12
xpack.security.transport.ssl.truststore.path: certs/elastic-certificates.p12
```

### 创建证书

```bash
# 创建自签名证书
./bin/elasticsearch-certutil ca
./bin/elasticsearch-certutil cert --ca elastic-stack-ca.p12
```

### 设置密码

```bash
# 自动设置密码
./bin/elasticsearch-setup-passwords auto

# 交互式设置密码
./bin/elasticsearch-setup-passwords interactive
```

## JVM 配置

### jvm.options

```bash
# 堆内存设置
-Xms8g
-Xmx8g

# 垃圾回收器
-XX:+UseG1GC

# GC日志
-XX:+PrintGCDetails
-XX:+PrintGCTimeStamps
-XX:+PrintGCDateStamps
-XX:+PrintHeapAtGC
-Xloggc:${path.logs}/gc.log

# CMS配置（Java 8）
# -XX:+UseConcMarkSweepGC
# -XX:CMSInitiatingOccupancyFraction=75
# -XX:+UseCMSInitiatingOccupancyOnly
```

### 堆内存建议

| 可用内存 | 堆内存设置 |
|----------|-----------|
| 4GB | -Xms2g -Xmx2g |
| 8GB | -Xms4g -Xmx4g |
| 16GB | -Xms8g -Xmx8g |
| 32GB | -Xms16g -Xmx16g |

## 监控与管理

### 健康检查

```bash
# 检查集群健康
curl -XGET 'http://localhost:9200/_cluster/health?pretty'

# 查看节点信息
curl -XGET 'http://localhost:9200/_nodes?pretty'

# 查看索引状态
curl -XGET 'http://localhost:9200/_cat/indices?v'
```

### 常用命令

```bash
# 查看集群状态
curl -XGET 'http://localhost:9200/_cluster/state?pretty'

# 查看分片分配
curl -XGET 'http://localhost:9200/_cat/shards?v'

# 手动分片分配
curl -XPOST 'http://localhost:9200/_cluster/reroute' \
  -H 'Content-Type: application/json' \
  -d '{
    "commands": [
      {
        "move": {
          "index": "my_index",
          "shard": 0,
          "from_node": "node-1",
          "to_node": "node-2"
        }
      }
    ]
  }'
```
