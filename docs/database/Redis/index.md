# Redis数据类型
| 数据类型     | 场景                                                      |
| ------------ | --------------------------------------------------------- |
| String       | 缓存、计数器、分布式锁、分布式Session、会话控制（cookie） |
| Hash         | 存储对象、购物车                                          |
| List（有序） | 消息队列、文章列表                                        |
| Set（无序）  | 标签、点赞、共同好友                                      |
| Zset（有序） | 排行榜、延时队列                                          |
| Bitmap       | 独立用户统计、布隆过滤器                                  |
| HyperLogLog  | 独立用户统计                                              |
| Geo          | 地理位置查询                                              |
