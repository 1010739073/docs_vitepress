# PHP 大厂面试题与高并发场景

## 一、PHP 基础原理

### 1. PHP 运行机制

**问题**：PHP 的运行机制是怎样的？

**详细答案**：
PHP 的运行机制包括以下几个阶段：

1. **词法分析**：
   - 将 PHP 代码分解为标记（tokens）
   - 识别关键字、变量、运算符等

2. **语法分析**：
   - 将标记组合成抽象语法树（AST）
   - 检查语法错误

3. **编译**：
   - 将 AST 编译为 opcode
   - opcode 是 Zend 虚拟机可执行的指令

4. **执行**：
   - Zend 虚拟机执行 opcode
   - 执行过程中可能涉及函数调用、变量操作等

5. **输出**：
   - 将执行结果输出到客户端

**代码示例**：

```php
<?php
// 简单的 PHP 程序
$name = "World";
echo "Hello, $name!\n";

// 函数定义
function greet($name) {
    return "Hello, $name!";
}

// 函数调用
echo greet("PHP") . "\n";
?>
```

### 2. PHP 内存管理

**问题**：PHP 的内存管理机制是怎样的？

**详细答案**：
PHP 的内存管理机制包括：

1. **引用计数**：
   - 每个变量都有一个引用计数
   - 引用计数为 0 时，内存被释放

2. **垃圾回收**：
   - 循环引用检测
   - 自动回收不可达的内存

3. **内存池**：
   - 预分配内存块
   - 减少内存分配和释放的开销

**代码示例**：

```php
<?php
// 引用计数示例
$a = "Hello"; // $a 的引用计数为 1
$b = $a;      // $a 和 $b 的引用计数都为 2
unset($a);    // $a 的引用计数为 1
unset($b);    // $b 的引用计数为 0，内存被释放

// 循环引用示例
class Node {
    public $next;
}

$node1 = new Node();
$node2 = new Node();
$node1->next = $node2;
$node2->next = $node1; // 循环引用

unset($node1);
unset($node2); // 垃圾回收器会自动回收

// 内存使用情况
echo memory_get_usage() . "\n";
echo memory_get_peak_usage() . "\n";
?>
```

### 3. PHP 错误处理

**问题**：PHP 的错误处理机制是怎样的？

**详细答案**：
PHP 的错误处理机制包括：

1. **错误级别**：
   - E_ERROR：致命错误
   - E_WARNING：警告
   - E_NOTICE：通知
   - E_DEPRECATED：废弃警告

2. **错误处理函数**：
   - set_error_handler()：设置自定义错误处理函数
   - set_exception_handler()：设置自定义异常处理函数
   - register_shutdown_function()：注册脚本结束时的函数

3. **异常处理**：
   - try-catch-finally 结构
   - throw 抛出异常
   - Exception 类及其子类

**代码示例**：

```php
<?php
// 自定义错误处理函数
function customErrorHandler($errno, $errstr, $errfile, $errline) {
    echo "Error: [$errno] $errstr in $errfile on line $errline\n";
    return true;
}

set_error_handler("customErrorHandler");

// 自定义异常处理函数
function customExceptionHandler($exception) {
    echo "Exception: " . $exception->getMessage() . "\n";
}

set_exception_handler("customExceptionHandler");

// 触发错误
echo $undefinedVariable; // 触发 E_NOTICE

// 抛出异常
try {
    throw new Exception("This is an exception");
} catch (Exception $e) {
    echo "Caught exception: " . $e->getMessage() . "\n";
} finally {
    echo "Finally block\n";
}
?>
```

```php
<?php
class ConnectionPool {
    private $pool = [];
    private $maxConnections = 10;
    private $currentConnections = 0;
    private $dsn;
    private $username;
    private $password;

    public function __construct($dsn, $username, $password, $maxConnections = 10) {
        $this->dsn = $dsn;
        $this->username = $username;
        $this->password = $password;
        $this->maxConnections = $maxConnections;
    }

    public function getConnection() {
        if (count($this->pool) > 0) {
            return array_pop($this->pool);
        }

        if ($this->currentConnections < $this->maxConnections) {
            $this->currentConnections++;
            return new PDO($this->dsn, $this->username, $this->password);
        }

        // 等待连接可用
        while (count($this->pool) === 0) {
            usleep(1000); // 等待 1ms
        }

        return array_pop($this->pool);
    }

    public function releaseConnection($connection) {
        if ($connection !== null) {
            array_push($this->pool, $connection);
        }
    }
}

// 使用示例
$pool = new ConnectionPool("mysql:host=localhost;dbname=test", "user", "password");

$connection = $pool->getConnection();
$stmt = $connection->prepare("SELECT * FROM users WHERE id = ?");
$stmt->execute([1]);
$user = $stmt->fetch(PDO::FETCH_ASSOC);

$pool->releaseConnection($connection);
?>
```

**代码示例（Redis 缓存）**：

```php
<?php
class CacheService {
    private $redis;

    public function __construct() {
        $this->redis = new Redis();
        $this->redis->connect('127.0.0.1', 6379);
    }

    public function get($key, $callback, $ttl = 3600) {
        $value = $this->redis->get($key);

        if ($value === false) {
            // 缓存未命中，调用回调函数获取数据
            $value = call_user_func($callback);

            // 将数据存入缓存
            $this->redis->setex($key, $ttl, serialize($value));
        } else {
            $value = unserialize($value);
        }

        return $value;
    }

    public function set($key, $value, $ttl = 3600) {
        return $this->redis->setex($key, $ttl, serialize($value));
    }

    public function delete($key) {
        return $this->redis->del($key);
    }
}

// 使用示例
$cache = new CacheService();

// 获取用户信息
$user = $cache->get('user:1', function() {
    // 模拟数据库查询
    return [
        'id' => 1,
        'name' => 'John',
        'email' => 'john@example.com',
    ];
});

var_dump($user);
?>
```

**代码示例（Redis 分布式锁）**：

```php
<?php
class DistributedLock {
    private $redis;
    private $lockTimeout = 10; // 锁超时时间（秒）

    public function __construct() {
        $this->redis = new Redis();
        $this->redis->connect('127.0.0.1', 6379);
    }

    public function acquire($key, $value) {
        // 使用 SET NX EX 命令获取锁
        $result = $this->redis->set($key, $value, ['NX', 'EX' => $this->lockTimeout]);
        return $result === true;
    }

    public function release($key, $value) {
        // 使用 Lua 脚本确保只有锁的持有者才能释放锁
        $script = "
            if redis.call('GET', KEYS[1]) == ARGV[1] then
                return redis.call('DEL', KEYS[1])
            else
                return 0
            end
        ";

        return $this->redis->eval($script, [$key, $value], 1) === 1;
    }
}

// 使用示例
$lock = new DistributedLock();
$lockKey = 'my_lock';
$lockValue = uniqid();

if ($lock->acquire($lockKey, $lockValue)) {
    try {
        // 执行业务逻辑
        echo "获取锁成功\n";
        sleep(2);
    } finally {
        $lock->release($lockKey, $lockValue);
        echo "释放锁\n";
    }
} else {
    echo "获取锁失败\n";
}
?>
```

```php
<?php
class RedisQueue {
    private $redis;

    public function __construct() {
        $this->redis = new Redis();
        $this->redis->connect('127.0.0.1', 6379);
    }

    public function push($queue, $data) {
        return $this->redis->rPush($queue, serialize($data));
    }

    public function pop($queue, $timeout = 0) {
        if ($timeout > 0) {
            $result = $this->redis->blPop($queue, $timeout);
            return $result ? unserialize($result[1]) : null;
        } else {
            $result = $this->redis->lPop($queue);
            return $result ? unserialize($result) : null;
        }
    }

    public function size($queue) {
        return $this->redis->lLen($queue);
    }
}

// 生产者
$queue = new RedisQueue();
$queue->push('task_queue', ['task' => 'send_email', 'data' => ['to' => 'user@example.com']]);
echo "任务已入队\n";

// 消费者
$queue = new RedisQueue();
while (true) {
    $task = $queue->pop('task_queue', 10);
    if ($task) {
        echo "处理任务: " . $task['task'] . "\n";
        // 处理任务逻辑
    } else {
        echo "队列为空，等待中...\n";

}
```

```php
// 输入过滤
$email = filter_input(INPUT_POST, 'email', FILTER_VALIDATE_EMAIL);
if ($email === false) {
    throw new InvalidArgumentException('Invalid email address');
}

// 去除 HTML 标签
$cleanInput = strip_tags($userInput);

// 设置 CSP 头
header("Content-Security-Policy: default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'");
?>
```

### 3. CSRF 防护

**问题**：如何防止 CSRF 攻击？

**详细答案**：
防止 CSRF 的方法：

1. **使用 CSRF Token**：
   - 生成随机的 CSRF Token
   - 在表单中包含 CSRF Token
   - 验证 CSRF Token

2. **检查 Referer 头**：
   - 验证请求来源
   - 拒绝非法来源的请求

3. **使用 SameSite Cookie**：
   - 设置 SameSite 属性
   - 防止跨站请求携带 Cookie

**代码示例**：

```php
<?php
session_start();

// 生成 CSRF Token
function generateCsrfToken() {
    if (empty($_SESSION['csrf_token'])) {
        $_SESSION['csrf_token'] = bin2hex(random_bytes(32));
    }
    return $_SESSION['csrf_token'];
}

// 验证 CSRF Token
function validateCsrfToken($token) {
    return isset($_SESSION['csrf_token']) && hash_equals($_SESSION['csrf_token'], $token);
}

// 在表单中包含 CSRF Token
$csrfToken = generateCsrfToken();
?>
<form method="POST" action="/submit">
    <input type="hidden" name="csrf_token" value="<?php echo $csrfToken; ?>">
    <input type="text" name="username">
    <button type="submit">Submit</button>
</form>

<?php
// 验证 CSRF Token
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $token = $_POST['csrf_token'] ?? '';
    if (!validateCsrfToken($token)) {
        die('CSRF token validation failed');
    }
    // 处理表单提交
}
?>
```
