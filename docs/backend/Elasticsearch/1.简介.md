# 1. 简介

## 什么是 Elasticsearch

Elasticsearch 是一个分布式、RESTful 风格的搜索和数据分析引擎，基于 Apache Lucene 构建。它提供了：

- **全文搜索**：强大的全文检索能力
- **实时分析**：毫秒级响应的数据分析
- **分布式架构**：水平扩展，高可用
- **RESTful API**：简单易用的接口

## 核心特点

### 分布式特性

| 特性 | 说明 |
|------|------|
| **水平扩展** | 通过添加节点轻松扩展容量 |
| **自动分片** | 数据自动分布到多个节点 |
| **负载均衡** | 请求自动路由到合适节点 |
| **故障转移** | 自动处理节点故障 |

### 搜索特性

| 特性 | 说明 |
|------|------|
| **全文搜索** | 支持复杂的文本搜索 |
| **结构化搜索** | 支持数值、日期等结构化数据 |
| **模糊搜索** | 支持拼写错误容忍 |
| **聚合分析** | 强大的数据分析能力 |

### 实时特性

| 特性 | 说明 |
|------|------|
| **近实时搜索** | 数据写入后立即可搜索 |
| **实时监控** | 集群状态实时监控 |
| **实时告警** | 基于条件的告警触发 |

## 应用场景

### 全文搜索
- 电商商品搜索
- 文档检索系统
- 日志分析平台

### 数据分析
- 业务指标分析
- 用户行为分析
- 实时监控仪表盘

### 日志管理
- 集中式日志存储
- 日志搜索分析
- 异常检测告警

### 安全分析
- 威胁检测
- 安全事件关联
- 合规审计

## 与其他技术对比

| 特性 | Elasticsearch | Solr | MySQL |
|------|---------------|------|-------|
| 全文搜索 | 原生支持 | 原生支持 | 有限支持 |
| 分布式 | 原生分布式 | 需要额外配置 | 需分库分表 |
| 实时性 | 近实时 | 近实时 | 实时 |
| 扩展性 | 水平扩展 | 较复杂 | 较复杂 |
| 聚合分析 | 强大 | 良好 | 有限 |
