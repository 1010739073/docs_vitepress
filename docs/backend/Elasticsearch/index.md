# Elasticsearch

> 分布式搜索与分析引擎 - 实时全文搜索、结构化搜索、分析和可视化

## 概述

Elasticsearch 是一个基于 Lucene 构建的分布式、RESTful 风格的搜索和数据分析引擎。它提供了一个分布式多用户能力的全文搜索引擎，能够快速地存储、搜索和分析大量数据。
