# Kafka 消费者详解

## 消费者架构

```
应用程序 → Consumer API → ConsumerCoordinator → 协调器（Broker） → 获取分区分配 → 从 Broker 拉取消息 → 提交 Offset
```

## 消费者组

### 消费者组概念

消费者组是一组消费者的集合，共同消费一个 Topic 的所有消息。

### 消费者组特点

1. **组内消息不重复** - 同一条消息只会被组内的一个消费者消费
2. **组间消息独立** - 不同消费者组可以独立消费相同的消息
3. **负载均衡** - Partition 均匀分配给组内的消费者
4. **容错性** - 消费者故障时，Partition 重新分配

### 消费者组图示

```
Topic: 3个Partition
Consumer Group A: 3个Consumer
├── Consumer 1 → Partition 0
├── Consumer 2 → Partition 1
└── Consumer 3 → Partition 2

Topic: 3个Partition
Consumer Group B: 2个Consumer
├── Consumer 1 → Partition 0, Partition 1
└── Consumer 2 → Partition 2

Topic: 3个Partition
Consumer Group C: 4个Consumer
├── Consumer 1 → Partition 0
├── Consumer 2 → Partition 1
├── Consumer 3 → Partition 2
└── Consumer 4 → 空闲
```

## 消费模式

### 1. 订阅模式（Subscribe）

订阅一个或多个 Topic，由 Kafka 自动分配 Partition。

```java
Properties props = new Properties();
props.put("bootstrap.servers", "localhost:9092");
props.put("group.id", "my-group");
props.put("key.deserializer", StringDeserializer.class.getName());
props.put("value.deserializer", StringDeserializer.class.getName());

KafkaConsumer<String, String> consumer = new KafkaConsumer<>(props);

// 订阅单个 Topic
consumer.subscribe(Collections.singletonList("my-topic"));

// 订阅多个 Topic
consumer.subscribe(Arrays.asList("topic1", "topic2", "topic3"));

// 使用正则表达式订阅
consumer.subscribe(Pattern.compile("topic-.*"));

// 消费消息
while (true) {
    ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));
    for (ConsumerRecord<String, String> record : records) {
        System.out.printf("partition=%d, offset=%d, key=%s, value=%s%n",
            record.partition(), record.offset(), record.key(), record.value());
    }
}
```

### 2. 手动分配模式（Assign）

手动指定要消费的 Partition，不使用消费者组。

```java
KafkaConsumer<String, String> consumer = new KafkaConsumer<>(props);

// 手动分配 Partition
List<TopicPartition> partitions = Arrays.asList(
    new TopicPartition("my-topic", 0),
    new TopicPartition("my-topic", 1)
);
consumer.assign(partitions);

// 消费消息
while (true) {
    ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));
    for (ConsumerRecord<String, String> record : records) {
        System.out.printf("partition=%d, offset=%d, key=%s, value=%s%n",
            record.partition(), record.offset(), record.key(), record.value());
    }
}
```

## Offset 管理

### 自动提交

```java
// 开启自动提交
props.put("enable.auto.commit", "true");
props.put("auto.commit.interval.ms", "5000");  // 每5秒提交一次

// 消费消息
while (true) {
    ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));
    for (ConsumerRecord<String, String> record : records) {
        // 处理消息
    }
    // Offset 自动提交
}
```

### 手动提交

#### 同步提交

```java
// 关闭自动提交
props.put("enable.auto.commit", "false");

while (true) {
    ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));

    try {
        for (ConsumerRecord<String, String> record : records) {
            // 处理消息
        }
        // 同步提交 Offset（阻塞直到提交成功）
        consumer.commitSync();
    } catch (CommitFailedException e) {
        System.err.println("Commit failed: " + e.getMessage());
    }
}
```

#### 异步提交

```java
// 关闭自动提交
props.put("enable.auto.commit", "false");

while (true) {
    ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));

    for (ConsumerRecord<String, String> record : records) {
        // 处理消息
    }
    // 异步提交 Offset（不阻塞）
    consumer.commitAsync(new OffsetCommitCallback() {
        @Override
        public void onComplete(Map<TopicPartition, OffsetAndMetadata> offsets,
                              Exception exception) {
            if (exception != null) {
                System.err.println("Commit failed: " + exception.getMessage());
            } else {
                System.out.println("Commit succeeded: " + offsets);
            }
        }
    });
}
```

#### 指定 Offset 提交

```java
// 提交特定 Partition 的 Offset
Map<TopicPartition, OffsetAndMetadata> offsets = new HashMap<>();
offsets.put(new TopicPartition("my-topic", 0), new OffsetAndMetadata(100));
offsets.put(new TopicPartition("my-topic", 1), new OffsetAndMetadata(200));

consumer.commitSync(offsets);
```

## Offset 重置策略

当消费者没有初始 Offset 或 Offset 无效时的重置策略。

```java
// earliest：从最早的消息开始消费
props.put("auto.offset.reset", "earliest");

// latest：从最新的消息开始消费（默认）
props.put("auto.offset.reset", "latest");

// none：抛出异常
props.put("auto.offset.reset", "none");
```

## 分区分配策略

### 范围分配（Range）

将 Partition 按范围分配给消费者。

```
Topic: 7个Partition，3个Consumer
Consumer 1: Partition 0, 1, 2
Consumer 2: Partition 3, 4, 5
Consumer 3: Partition 6
```

```java
props.put("partition.assignment.strategy",
    "org.apache.kafka.clients.consumer.RangeAssignor");
```

### 轮询分配（RoundRobin）

轮询将 Partition 分配给消费者。

```
Topic: 7个Partition，3个Consumer
Consumer 1: Partition 0, 3, 6
Consumer 2: Partition 1, 4
Consumer 3: Partition 2, 5
```

```java
props.put("partition.assignment.strategy",
    "org.apache.kafka.clients.consumer.RoundRobinAssignor");
```

### 黏性分配（Sticky）

在保证分配均匀的同时，尽量保留之前的分配关系，减少分区迁移。

```java
props.put("partition.assignment.strategy",
    "org.apache.kafka.clients.consumer.StickyAssignor");
```

### 自定义分配策略

```java
public class CustomAssignor implements ConsumerPartitionAssignor {
    @Override
    public String name() {
        return "custom-assignor";
    }

    @Override
    public PartitionAssignor.Assignment assignment(Subscription subscription) {
        // 自定义分配逻辑
        return new PartitionAssignor.Assignment(assignments);
    }

    @Override
    public Subscription subscription(Set<String> topics) {
        return new Subscription(new ArrayList<>(topics));
    }

    @Override
    public void onAssignment(Assignment assignment) {
        // 收到分配后的回调
    }
}
```

## 消费者配置详解

```java
Properties props = new Properties();

// 基本配置
props.put("bootstrap.servers", "localhost:9092");
props.put("group.id", "my-group");
props.put("key.deserializer", StringDeserializer.class.getName());
props.put("value.deserializer", StringDeserializer.class.getName());

// Offset 管理
props.put("enable.auto.commit", "false");
props.put("auto.offset.reset", "latest");

// 会话和心跳
props.put("session.timeout.ms", 10000);        // 会话超时时间
props.put("heartbeat.interval.ms", 3000);       // 心跳间隔时间
props.put("max.poll.interval.ms", 300000);      // 最大拉取间隔

// 拉取配置
props.put("max.poll.records", 500);             // 每次拉取的最大记录数
props.put("fetch.min.bytes", 1);                // 最小拉取字节数
props.put("fetch.max.wait.ms", 500);            // 最大等待时间
props.put("fetch.max.bytes", 52428800);         // 最大拉取字节数

// 网络配置
props.put("connections.max.idle.ms", 540000);   // 连接最大空闲时间
props.put("request.timeout.ms", 30000);         // 请求超时时间
```

## 消费者 Rebalance

Rebalance 是指消费者组中的消费者数量变化时，Partition 重新分配的过程。

### Rebalance 触发条件

1. 消费者加入或离开组
2. 消费者会话超时
3. 订阅的 Topic 或 Partition 数量变化

### Rebalance 监听器

```java
props.put("group.instance.id", "my-consumer-1");  // 静态成员 ID

consumer.subscribe(Collections.singletonList("my-topic"),
    new ConsumerRebalanceListener() {
        @Override
        public void onPartitionsRevoked(Collection<TopicPartition> partitions) {
            // 分区被收回前调用
            System.out.println("Partitions revoked: " + partitions);
            // 可以在这里提交 Offset
            consumer.commitSync();
        }

        @Override
        public void onPartitionsAssigned(Collection<TopicPartition> partitions) {
            // 分区分配后调用
            System.out.println("Partitions assigned: " + partitions);
            // 可以在这里重置 Offset
            for (TopicPartition partition : partitions) {
                consumer.seek(partition, 0);
            }
        }
    });
```

## 消费者多线程

### 单消费者多线程处理

```java
public class ConsumerThreadExample {
    private final KafkaConsumer<String, String> consumer;
    private final ExecutorService executor;

    public ConsumerThreadExample() {
        Properties props = new Properties();
        props.put("bootstrap.servers", "localhost:9092");
        props.put("group.id", "multi-thread-group");
        props.put("key.deserializer", StringDeserializer.class.getName());
        props.put("value.deserializer", StringDeserializer.class.getName());
        props.put("enable.auto.commit", "false");

        this.consumer = new KafkaConsumer<>(props);
        this.executor = Executors.newFixedThreadPool(10);
    }

    public void consume() {
        consumer.subscribe(Collections.singletonList("my-topic"));

        try {
            while (true) {
                ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));

                // 为每条消息创建一个处理任务
                for (ConsumerRecord<String, String> record : records) {
                    executor.submit(() -> processMessage(record));
                }

                // 手动提交 Offset
                consumer.commitAsync();
            }
        } finally {
            consumer.close();
            executor.shutdown();
        }
    }

    private void processMessage(ConsumerRecord<String, String> record) {
        // 处理消息
        System.out.println("Processing: " + record.value());
    }
}
```

### 多消费者多线程

```java
public class MultiConsumerExample {
    private final int consumerCount = 3;

    public void start() {
        ExecutorService executor = Executors.newFixedThreadPool(consumerCount);

        for (int i = 0; i < consumerCount; i++) {
            final int index = i;
            executor.submit(() -> {
                Properties props = new Properties();
                props.put("bootstrap.servers", "localhost:9092");
                props.put("group.id", "multi-consumer-group");
                props.put("key.deserializer", StringDeserializer.class.getName());
                props.put("value.deserializer", StringDeserializer.class.getName());
                props.put("enable.auto.commit", "false");

                KafkaConsumer<String, String> consumer = new KafkaConsumer<>(props);
                consumer.subscribe(Collections.singletonList("my-topic"));

                try {
                    while (true) {
                        ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));

                        for (ConsumerRecord<String, String> record : records) {
                            System.out.printf("Consumer %d: %s%n", index, record.value());
                        }

                        consumer.commitAsync();
                    }
                } finally {
                    consumer.close();
                }
            });
        }
    }
}
```

## 完整示例

```java
import org.apache.kafka.clients.consumer.*;
import org.apache.kafka.common.TopicPartition;
import org.apache.kafka.common.serialization.StringDeserializer;

import java.time.Duration;
import java.util.*;
import java.util.concurrent.*;

public class KafkaConsumerExample {
    public static void main(String[] args) {
        // 配置
        Properties props = new Properties();
        props.put("bootstrap.servers", "localhost:9092");
        props.put("group.id", "my-group");
        props.put("key.deserializer", StringDeserializer.class.getName());
        props.put("value.deserializer", StringDeserializer.class.getName());

        // Offset 配置
        props.put("enable.auto.commit", "false");
        props.put("auto.offset.reset", "earliest");

        // 会话配置
        props.put("session.timeout.ms", 10000);
        props.put("heartbeat.interval.ms", 3000);

        // 拉取配置
        props.put("max.poll.records", 500);
        props.put("max.poll.interval.ms", 300000);

        // 创建消费者
        KafkaConsumer<String, String> consumer = new KafkaConsumer<>(props);

        // 订阅 Topic
        consumer.subscribe(Collections.singletonList("my-topic"),
            new ConsumerRebalanceListener() {
                @Override
                public void onPartitionsRevoked(Collection<TopicPartition> partitions) {
                    System.out.println("Partitions revoked: " + partitions);
                    consumer.commitSync();
                }

                @Override
                public void onPartitionsAssigned(Collection<TopicPartition> partitions) {
                    System.out.println("Partitions assigned: " + partitions);
                }
            });

        // 消费消息
        try {
            while (true) {
                ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));

                for (ConsumerRecord<String, String> record : records) {
                    System.out.printf("partition=%d, offset=%d, key=%s, value=%s%n",
                        record.partition(), record.offset(), record.key(), record.value());

                    // 处理消息...
                }

                // 手动提交 Offset
                consumer.commitAsync((offsets, exception) -> {
                    if (exception != null) {
                        System.err.println("Commit failed: " + exception.getMessage());
                    }
                });
            }
        } catch (Exception e) {
            e.printStackTrace();
        } finally {
            consumer.close();
        }
    }
}
```
