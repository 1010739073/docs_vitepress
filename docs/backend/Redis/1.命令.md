# Redis常用命令

Redis是一个开源的内存数据库，支持多种数据类型。本文档按数据类型分类整理了Redis的常用命令。

## 一、字符串类型（String）

| 命令   | 语法                               | 说明                             |
| :----- | :--------------------------------- | :------------------------------- |
| SET    | `SET key value`                    | 设置键值对                       |
| GET    | `GET key`                          | 获取键对应的值                   |
| GETSET | `GETSET key value`                 | 设置新值并返回旧值               |
| MSET   | `MSET key1 value1 key2 value2 ...` | 同时设置多个键值对               |
| MGET   | `MGET key1 key2 ...`               | 同时获取多个键的值               |
| INCR   | `INCR key`                         | 将键的值加1                      |
| INCRBY | `INCRBY key increment`             | 将键的值增加指定的增量           |
| DECR   | `DECR key`                         | 将键的值减1                      |
| DECRBY | `DECRBY key decrement`             | 将键的值减少指定的减量           |
| APPEND | `APPEND key value`                 | 向字符串末尾追加内容             |
| STRLEN | `STRLEN key`                       | 获取字符串的长度                 |
| SETEX  | `SETEX key seconds value`          | 设置键值对并指定过期时间         |
| SETNX  | `SETNX key value`                  | 仅当键不存在时设置键值对         |
| PSETEX | `PSETEX key milliseconds value`    | 设置键值对并指定过期时间（毫秒） |

## 二、列表类型（List）

| 命令   | 语法                          | 说明                             |
| :----- | :---------------------------- | :------------------------------- |
| LPUSH  | `LPUSH key value1 value2 ...` | 向列表左侧添加一个或多个元素     |
| RPUSH  | `RPUSH key value1 value2 ...` | 向列表右侧添加一个或多个元素     |
| LPOP   | `LPOP key`                    | 移除并返回列表左侧的第一个元素   |
| RPOP   | `RPOP key`                    | 移除并返回列表右侧的第一个元素   |
| LLEN   | `LLEN key`                    | 获取列表的长度                   |
| LRANGE | `LRANGE key start stop`       | 获取列表中指定范围内的元素       |
| LINDEX | `LINDEX key index`            | 获取列表中指定索引的元素         |
| LSET   | `LSET key index value`        | 设置列表中指定索引的元素值       |
| LREM   | `LREM key count value`        | 移除列表中指定值的元素           |
| LTRIM  | `LTRIM key start stop`        | 修剪列表，只保留指定范围内的元素 |
| BLPOP  | `BLPOP key1 key2 ... timeout` | 阻塞式弹出列表左侧的元素         |
| BRPOP  | `BRPOP key1 key2 ... timeout` | 阻塞式弹出列表右侧的元素         |

## 三、哈希类型（Hash）

| 命令         | 语法                                        | 说明                               |
| :----------- | :------------------------------------------ | :--------------------------------- |
| HSET         | `HSET key field value`                      | 设置哈希表字段的值                 |
| HGET         | `HGET key field`                            | 获取哈希表字段的值                 |
| HMSET        | `HMSET key field1 value1 field2 value2 ...` | 同时设置多个哈希表字段的值         |
| HMGET        | `HMGET key field1 field2 ...`               | 同时获取多个哈希表字段的值         |
| HGETALL      | `HGETALL key`                               | 获取哈希表所有字段和值             |
| HDEL         | `HDEL key field1 field2 ...`                | 删除哈希表的一个或多个字段         |
| HLEN         | `HLEN key`                                  | 获取哈希表的字段数量               |
| HEXISTS      | `HEXISTS key field`                         | 检查哈希表是否存在指定字段         |
| HINCRBY      | `HINCRBY key field increment`               | 将哈希表字段的值增加指定的增量     |
| HINCRBYFLOAT | `HINCRBYFLOAT key field increment`          | 将哈希表字段的值增加指定的浮点增量 |
| HKEYS        | `HKEYS key`                                 | 获取哈希表所有字段名               |
| HVALS        | `HVALS key`                                 | 获取哈希表所有字段值               |

## 四、集合类型（Set）

| 命令        | 语法                                    | 说明                                 |
| :---------- | :-------------------------------------- | :----------------------------------- |
| SADD        | `SADD key member1 member2 ...`          | 向集合添加一个或多个成员             |
| SREM        | `SREM key member1 member2 ...`          | 从集合移除一个或多个成员             |
| SMEMBERS    | `SMEMBERS key`                          | 获取集合所有成员                     |
| SISMEMBER   | `SISMEMBER key member`                  | 检查成员是否存在于集合中             |
| SCARD       | `SCARD key`                             | 获取集合的成员数量                   |
| SRANDMEMBER | `SRANDMEMBER key [count]`               | 随机返回集合中的一个或多个成员       |
| SPOP        | `SPOP key [count]`                      | 随机移除并返回集合中的一个或多个成员 |
| SDIFF       | `SDIFF key1 key2 ...`                   | 计算集合的差集                       |
| SDIFFSTORE  | `SDIFFSTORE destination key1 key2 ...`  | 计算集合的差集并存储到新集合         |
| SINTER      | `SINTER key1 key2 ...`                  | 计算集合的交集                       |
| SINTERSTORE | `SINTERSTORE destination key1 key2 ...` | 计算集合的交集并存储到新集合         |
| SUNION      | `SUNION key1 key2 ...`                  | 计算集合的并集                       |
| SUNIONSTORE | `SUNIONSTORE destination key1 key2 ...` | 计算集合的并集并存储到新集合         |

## 五、有序集合类型（Sorted Set）

| 命令             | 语法                                                             | 说明                                             |
| :--------------- | :--------------------------------------------------------------- | :----------------------------------------------- |
| ZADD             | `ZADD key score1 member1 score2 member2 ...`                     | 向有序集合添加一个或多个成员                     |
| ZREM             | `ZREM key member1 member2 ...`                                   | 从有序集合移除一个或多个成员                     |
| ZSCORE           | `ZSCORE key member`                                              | 获取有序集合中成员的分数                         |
| ZINCRBY          | `ZINCRBY key increment member`                                   | 增加有序集合中成员的分数                         |
| ZCARD            | `ZCARD key`                                                      | 获取有序集合的成员数量                           |
| ZCOUNT           | `ZCOUNT key min max`                                             | 计算有序集合中分数在指定范围内的成员数量         |
| ZRANK            | `ZRANK key member`                                               | 获取有序集合中成员的排名（从小到大）             |
| ZREVRANK         | `ZREVRANK key member`                                            | 获取有序集合中成员的排名（从大到小）             |
| ZRANGE           | `ZRANGE key start stop [WITHSCORES]`                             | 获取有序集合中指定范围内的成员（从小到大）       |
| ZREVRANGE        | `ZREVRANGE key start stop [WITHSCORES]`                          | 获取有序集合中指定范围内的成员（从大到小）       |
| ZRANGEBYSCORE    | `ZRANGEBYSCORE key min max [WITHSCORES] [LIMIT offset count]`    | 获取有序集合中分数在指定范围内的成员             |
| ZREVRANGEBYSCORE | `ZREVRANGEBYSCORE key max min [WITHSCORES] [LIMIT offset count]` | 获取有序集合中分数在指定范围内的成员（从大到小） |
| ZREMRANGEBYRANK  | `ZREMRANGEBYRANK key start stop`                                 | 移除有序集合中指定排名范围内的成员               |
| ZREMRANGEBYSCORE | `ZREMRANGEBYSCORE key min max`                                   | 移除有序集合中分数在指定范围内的成员             |

## 六、通用命令

| 命令       | 语法                                   | 说明                         |
| :--------- | :------------------------------------- | :--------------------------- |
| KEYS       | `KEYS pattern`                         | 查找所有匹配指定模式的键     |
| EXISTS     | `EXISTS key1 key2 ...`                 | 检查一个或多个键是否存在     |
| DEL        | `DEL key1 key2 ...`                    | 删除一个或多个键             |
| TYPE       | `TYPE key`                             | 获取键的数据类型             |
| EXPIRE     | `EXPIRE key seconds`                   | 设置键的过期时间             |
| PEXPIRE    | `PEXPIRE key milliseconds`             | 设置键的过期时间（毫秒）     |
| EXPIREAT   | `EXPIREAT key timestamp`               | 设置键的过期时间戳           |
| PEXPIREAT  | `PEXPIREAT key milliseconds-timestamp` | 设置键的过期时间戳（毫秒）   |
| TTL        | `TTL key`                              | 获取键的剩余过期时间         |
| PTTL       | `PTTL key`                             | 获取键的剩余过期时间（毫秒） |
| PERSIST    | `PERSIST key`                          | 移除键的过期时间             |
| RENAME     | `RENAME key newkey`                    | 重命名键                     |
| RENAMENX   | `RENAMENX key newkey`                  | 仅当新键不存在时重命名键     |
| MOVE       | `MOVE key db`                          | 将键移动到指定数据库         |
| FLUSHDB    | `FLUSHDB`                              | 清空当前数据库               |
| FLUSHALL   | `FLUSHALL`                             | 清空所有数据库               |
| DBSIZE     | `DBSIZE`                               | 获取当前数据库中的键数量     |
| SELECT     | `SELECT index`                         | 切换到指定数据库             |
| INFO       | `INFO [section]`                       | 获取Redis服务器的信息        |
| CONFIG GET | `CONFIG GET parameter`                 | 获取Redis配置参数            |
| CONFIG SET | `CONFIG SET parameter value`           | 设置Redis配置参数            |
| SAVE       | `SAVE`                                 | 同步保存数据到磁盘           |
| BGSAVE     | `BGSAVE`                               | 异步保存数据到磁盘           |
| LASTSAVE   | `LASTSAVE`                             | 获取最后一次保存的时间戳     |
| SHUTDOWN   | `SHUTDOWN [SAVE                        | NOSAVE]`                     | 关闭Redis服务器 |

## 七、发布/订阅命令

| 命令         | 语法                                   | 说明                       |
| :----------- | :------------------------------------- | :------------------------- |
| PUBLISH      | `PUBLISH channel message`              | 向指定频道发布消息         |
| SUBSCRIBE    | `SUBSCRIBE channel1 channel2 ...`      | 订阅一个或多个频道         |
| UNSUBSCRIBE  | `UNSUBSCRIBE [channel1 channel2 ...]`  | 取消订阅一个或多个频道     |
| PSUBSCRIBE   | `PSUBSCRIBE pattern1 pattern2 ...`     | 订阅匹配指定模式的频道     |
| PUNSUBSCRIBE | `PUNSUBSCRIBE [pattern1 pattern2 ...]` | 取消订阅匹配指定模式的频道 |

## 八、事务命令

| 命令    | 语法                  | 说明                       |
| :------ | :-------------------- | :------------------------- |
| MULTI   | `MULTI`               | 开始事务                   |
| EXEC    | `EXEC`                | 执行事务                   |
| DISCARD | `DISCARD`             | 取消事务                   |
| WATCH   | `WATCH key1 key2 ...` | 监视一个或多个键，用于事务 |
| UNWATCH | `UNWATCH`             | 取消监视所有键             |

## 九、脚本命令

| 命令          | 语法                                               | 说明                        |
| :------------ | :------------------------------------------------- | :-------------------------- |
| EVAL          | `EVAL script numkeys key [key ...] arg [arg ...]`  | 执行Lua脚本                 |
| EVALSHA       | `EVALSHA sha1 numkeys key [key ...] arg [arg ...]` | 通过脚本的SHA1值执行Lua脚本 |
| SCRIPT LOAD   | `SCRIPT LOAD script`                               | 加载Lua脚本到内存           |
| SCRIPT EXISTS | `SCRIPT EXISTS sha1 [sha1 ...]`                    | 检查脚本是否存在于内存中    |
| SCRIPT FLUSH  | `SCRIPT FLUSH`                                     | 清空所有脚本缓存            |
| SCRIPT KILL   | `SCRIPT KILL`                                      | 终止当前正在执行的脚本      |
