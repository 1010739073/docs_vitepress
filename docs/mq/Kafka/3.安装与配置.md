# Kafka 安装与配置

## 环境要求

- Java 8 或更高版本
- ZooKeeper 3.4.x 或更高版本（Kafka 3.0+ 可以使用 KRaft 模式，无需 ZooKeeper）
- Linux/macOS/Windows 系统

## 安装步骤

### 1. 下载 Kafka

```bash
# 下载最新版本
wget https://downloads.apache.org/kafka/3.7.0/kafka_2.13-3.7.0.tgz

# 解压
tar -xzf kafka_2.13-3.7.0.tgz
cd kafka_2.13-3.7.0
```

### 2. 启动 ZooKeeper（传统模式）

```bash
# 启动 ZooKeeper
bin/zookeeper-server-start.sh config/zookeeper.properties

# 或使用后台模式
bin/zookeeper-server-start.sh -daemon config/zookeeper.properties
```

### 3. 启动 Kafka Broker

```bash
# 启动 Kafka
bin/kafka-server-start.sh config/server.properties

# 或使用后台模式
bin/kafka-server-start.sh -daemon config/server.properties
```

### 4. 使用 KRaft 模式（无需 ZooKeeper）

```bash
# 生成存储目录 UUID
bin/kafka-storage.sh random-uuid

# 格式化存储目录
bin/kafka-storage.sh format -t <UUID> -c config/kraft/server.properties

# 启动 Kafka
bin/kafka-server-start.sh config/kraft/server.properties
```

## 基本操作

### 创建 Topic

```bash
bin/kafka-topics.sh --create \
  --bootstrap-server localhost:9092 \
  --topic my-topic \
  --partitions 3 \
  --replication-factor 1
```

### 查看 Topic 列表

```bash
bin/kafka-topics.sh --list --bootstrap-server localhost:9092
```

### 查看 Topic 详情

```bash
bin/kafka-topics.sh --describe \
  --bootstrap-server localhost:9092 \
  --topic my-topic
```

### 修改 Topic

```bash
# 增加分区
bin/kafka-topics.sh --alter \
  --bootstrap-server localhost:9092 \
  --topic my-topic \
  --partitions 6

# 修改配置
bin/kafka-configs.sh --alter \
  --bootstrap-server localhost:9092 \
  --entity-type topics \
  --entity-name my-topic \
  --add-config retention.ms=86400000
```

### 删除 Topic

```bash
bin/kafka-topics.sh --delete \
  --bootstrap-server localhost:9092 \
  --topic my-topic
```

## 生产消息

```bash
# 命令行生产者
bin/kafka-console-producer.sh \
  --bootstrap-server localhost:9092 \
  --topic my-topic

# 输入消息后按回车发送
```

## 消费消息

```bash
# 从最新消息开始消费
bin/kafka-console-consumer.sh \
  --bootstrap-server localhost:9092 \
  --topic my-topic

# 从最早消息开始消费
bin/kafka-console-consumer.sh \
  --bootstrap-server localhost:9092 \
  --topic my-topic \
  --from-beginning

# 指定消费者组
bin/kafka-console-consumer.sh \
  --bootstrap-server localhost:9092 \
  --topic my-topic \
  --group my-group
```

## 核心配置文件

### server.properties（Broker 配置）

```properties
# Broker ID，集群内必须唯一
broker.id=0

# 监听地址和端口
listeners=PLAINTEXT://:9092
advertised.listeners=PLAINTEXT://localhost:9092

# 日志目录
log.dirs=/tmp/kafka-logs

# ZooKeeper 连接地址
zookeeper.connect=localhost:2181

# 默认分区数
num.partitions=1

# 默认副本数
default.replication.factor=1

# 日志保留时间（小时）
log.retention.hours=168

# 日志保留大小（字节）
log.retention.bytes=1073741824

# 日志段大小（字节）
log.segment.bytes=1073741824

# 日志检查间隔（毫秒）
log.retention.check.interval.ms=300000

# 副本拉取超时时间（毫秒）
replica.lag.time.max.ms=10000

# 最小同步副本数
min.insync.replicas=1

# 是否启用自动创建 Topic
auto.create.topics.enable=true

# 是否启用自动删除 Topic
auto.delete.topics.enable=false

# 是否启用日志压缩
log.cleaner.enable=false

# 网络线程数
num.network.threads=3

# I/O 线程数
num.io.threads=8

# socket 发送缓冲区大小
socket.send.buffer.bytes=102400

# socket 接收缓冲区大小
socket.receive.buffer.bytes=102400

# socket 请求最大字节数
socket.request.max.bytes=104857600
```

### producer.properties（生产者配置）

```properties
# Bootstrap 服务器列表
bootstrap.servers=localhost:9092

# Key 序列化器
key.serializer=org.apache.kafka.common.serialization.StringSerializer

# Value 序列化器
value.serializer=org.apache.kafka.common.serialization.StringSerializer

# 确认机制（0, 1, all/-1）
acks=1

# 重试次数
retries=0

# 批量发送大小（字节）
batch.size=16384

# 批量发送延迟（毫秒）
linger.ms=0

# 缓冲区大小（字节）
buffer.memory=33554432

# 压缩类型（none, gzip, snappy, lz4, zstd）
compression.type=none

# 发送超时时间（毫秒）
request.timeout.ms=30000
```

### consumer.properties（消费者配置）

```properties
# Bootstrap 服务器列表
bootstrap.servers=localhost:9092

# 消费者组 ID
group.id=test-consumer-group

# Key 反序列化器
key.deserializer=org.apache.kafka.common.serialization.StringDeserializer

# Value 反序列化器
value.deserializer=org.apache.kafka.common.serialization.StringDeserializer

# 是否自动提交 offset
enable.auto.commit=true

# 自动提交间隔（毫秒）
auto.commit.interval.ms=1000

# offset 重置策略（earliest, latest, none）
auto.offset.reset=latest

# 会话超时时间（毫秒）
session.timeout.ms=10000

# 心跳间隔时间（毫秒）
heartbeat.interval.ms=3000

# 最大拉取记录数
max.poll.records=500

# 最大拉取间隔（毫秒）
max.poll.interval.ms=300000
```

## 集群配置

### 多 Broker 集群

#### Broker 1 配置

```properties
broker.id=1
listeners=PLAINTEXT://:9092
advertised.listeners=PLAINTEXT://192.168.1.100:9092
log.dirs=/tmp/kafka-logs-1
zookeeper.connect=192.168.1.100:2181,192.168.1.101:2181,192.168.1.102:2181
```
