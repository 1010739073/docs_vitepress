# RabbitMQ 高级特性

## 一、死信队列（DLX）

### 1.1 什么是死信

消息成为死信的情况：

| 情况 | 说明 |
|------|------|
| **消息被拒绝** | 消费者调用 `basicNack` 或 `basicReject`，且 `requeue=false` |
| **消息过期** | 消息 TTL 到期 |
| **队列满** | 队列达到最大长度 |
| **队列删除** | 队列被删除 |

### 1.2 配置死信队列

```java
@Configuration
public class DeadLetterConfig {

    // 1. 声明死信交换机和队列
    @Bean
    public DirectExchange dlxExchange() {
        return new DirectExchange("dlx.exchange");
    }

    @Bean
    public Queue dlxQueue() {
        return QueueBuilder.durable("dlx.queue").build();
    }

    @Bean
    public Binding dlxBinding() {
        return BindingBuilder.bind(dlxQueue())
            .to(dlxExchange())
            .with("dlx.routing.key");
    }

    // 2. 声明业务队列，绑定死信交换机
    @Bean
    public Queue businessQueue() {
        Map<String, Object> args = new HashMap<>();
        args.put("x-dead-letter-exchange", "dlx.exchange");
        args.put("x-dead-letter-routing-key", "dlx.routing.key");
        args.put("x-message-ttl", 60000); // 消息 TTL 60 秒

        return QueueBuilder.durable("business.queue")
            .withArguments(args)
            .build();
    }
}
```

### 1.4 死信消息处理

```java
@Service
public class DeadLetterHandler {

    @Autowired
    private RabbitTemplate rabbitTemplate;

    /**
     * 手动处理死信消息
     */
    public void handleDeadLetter(String messageId) {
        // 1. 从数据库获取死信消息
        DeadLetterMessage message = getDeadLetterMessage(messageId);

        // 2. 根据原因处理
        String reason = message.getReason();

        switch (reason) {
            case "rejected":
                // 被拒绝，可能需要修复数据后重试
                retryAfterFix(message);
                break;
            case "expired":
                // 过期，可能需要取消相关业务
                cancelRelatedBusiness(message);
                break;
            case "maxlen":
                // 队列满，需要扩容或清理
                handleQueueFull(message);
                break;
            default:
                log.warn("未知死信原因: {}", reason);
        }

        // 3. 标记为已处理
        markAsProcessed(messageId);
    }
}
```

## 二、延迟队列

### 2.1 基于死信队列实现延迟队列

```java
@Configuration
public class DelayQueueConfig {

    // 1. 声明延迟队列（TTL 队列）
    @Bean
    public Queue delayQueue() {
        Map<String, Object> args = new HashMap<>();
        args.put("x-message-ttl", 30000); // 30 秒
        args.put("x-dead-letter-exchange", "dlx.exchange");
        args.put("x-dead-letter-routing-key", "target.routing.key");

        return QueueBuilder.durable("delay.queue")
            .withArguments(args)
            .build();
    }

    // 2. 声明目标队列
    @Bean
    public Queue targetQueue() {
        return QueueBuilder.durable("target.queue").build();
    }

    // 3. 声明交换机和绑定
    @Bean
    public DirectExchange dlxExchange() {
        return new DirectExchange("dlx.exchange");
    }

    @Bean
    public Binding targetBinding() {
        return BindingBuilder.bind(targetQueue())
            .to(dlxExchange())
            .with("target.routing.key");
    }
}
```

### 2.2 使用延迟队列

```java
@Service
public class DelayQueueService {

    @Autowired
    private RabbitTemplate rabbitTemplate;

    /**
     * 发送延迟消息
     * @param message 消息内容
     * @param delayMillis 延迟时间（毫秒）
     */
    public void sendDelayedMessage(Object message, long delayMillis) {
        Map<String, Object> headers = new HashMap<>();
        headers.put("x-delay", delayMillis);

        Message msg = MessageBuilder
            .withBody(JSON.toJSONBytes(message))
            .copyHeaders(headers)
            .build();

        rabbitTemplate.send("", "delay.queue", msg);
    }

    /**
     * 订单超时取消
     */
    public void scheduleOrderCancel(Long orderId, long delayMinutes) {
        long delayMillis = delayMinutes * 60 * 1000;

        Map<String, Object> message = new HashMap<>();
        message.put("orderId", orderId);
        message.put("action", "cancel");

        sendDelayedMessage(message, delayMillis);
    }
}

// 消费延迟消息
@RabbitListener(queues = "target.queue")
public void handleDelayedMessage(Map<String, Object> message) {
    String action = (String) message.get("action");

    if ("cancel".equals(action)) {
        Long orderId = ((Number) message.get("orderId")).longValue();
        // 取消订单
        orderService.cancelOrder(orderId);
    }
}
```

### 2.3 使用延迟插件

**安装延迟插件**：
```bash
rabbitmq-plugins enable rabbitmq_delayed_message_exchange
```

**配置延迟交换机**：
```java
@Configuration
public class DelayedExchangeConfig {

    @Bean
    public CustomExchange delayedExchange() {
        Map<String, Object> args = new HashMap<>();
        args.put("x-delayed-type", "direct");

        return new CustomExchange("delayed.exchange", "x-delayed-message", true, false, args);
    }

    @Bean
    public Queue delayedQueue() {
        return QueueBuilder.durable("delayed.queue").build();
    }

    @Bean
    public Binding delayedBinding() {
        return BindingBuilder.bind(delayedQueue())
            .to(delayedExchange())
            .with("delayed.routing.key")
            .noargs();
    }
}
```

**发送延迟消息**：
```java
@Service
public class DelayedMessageService {

    @Autowired
    private RabbitTemplate rabbitTemplate;

    public void sendDelayedMessage(Object message, long delayMillis) {
        Message msg = MessageBuilder
            .withBody(JSON.toJSONBytes(message))
            .setHeader("x-delay", delayMillis)
            .build();

        rabbitTemplate.convertAndSend("delayed.exchange", "delayed.routing.key", msg);
    }
}
```

```java
@Service
public class PriorityMessageService {

    @Autowired
    private RabbitTemplate rabbitTemplate;

    /**
     * 发送优先级消息
     * @param message 消息内容
     * @param priority 优先级（0-9，数字越大优先级越高）
     */
    public void sendPriorityMessage(Object message, int priority) {
        Message msg = MessageBuilder
            .withBody(JSON.toJSONBytes(message))
            .setPriority(priority)
            .build();

        rabbitTemplate.convertAndSend("", "priority.queue", msg);
    }

    /**
     * VIP 用户处理优先
     */
    public void processUserRequest(UserRequest request) {
        int priority = request.isVip() ? 9 : 5;
        sendPriorityMessage(request, priority);
    }
}
```

## 四、消息去重

### 4.1 基于 Redis 去重

```java
@Service
public class MessageDeduplicator {

    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    /**
     * 检查消息是否已处理
     */
    public boolean isProcessed(String messageId) {
        String key = "processed:" + messageId;
        return Boolean.TRUE.equals(redisTemplate.hasKey(key));
    }

    /**
     * 标记消息已处理
     */
    public void markAsProcessed(String messageId, long ttl) {
        String key = "processed:" + messageId;
        redisTemplate.opsForValue().set(key, "1", ttl, TimeUnit.SECONDS);
    }

    /**
     * 幂等消费
     */
    @RabbitListener(queues = "my.queue")
    public void idempotentConsume(Message message) {
        String messageId = message.getMessageProperties().getMessageId();

        if (isProcessed(messageId)) {
            log.info("消息已处理，跳过: {}", messageId);
            return;
        }

        try {
            // 处理消息
            processMessage(message);

            // 标记已处理（保留 24 小时）
            markAsProcessed(messageId, 24 * 3600);

        } catch (Exception e) {
            log.error("处理失败", e);
            throw e;
        }
    }
}
```

### 4.2 基于数据库去重

```java
@Service
public class DatabaseDeduplicator {

    @Autowired
    private MessageLogMapper messageLogMapper;

    /**
     * 检查消息是否已处理
     */
    public boolean isProcessed(String messageId) {
        MessageLog log = messageLogMapper.selectById(messageId);
        return log != null && "SUCCESS".equals(log.getStatus());
    }

    /**
     * 标记消息已处理
     */
    public void markAsProcessed(String messageId) {
        MessageLog log = new MessageLog();
        log.setMessageId(messageId);
        log.setStatus("SUCCESS");
        log.setProcessTime(new Date());
        messageLogMapper.insert(log);
    }
}
```

### 4.3 基于布隆过滤器去重

```java
@Service
public class BloomFilterDeduplicator {

    private final BloomFilter<String> bloomFilter;

    public BloomFilterDeduplicator() {
        // 预期元素数量 100 万，误判率 0.01%
        this.bloomFilter = BloomFilter.create(Funnels.stringFunnel(Charset.defaultCharset()),
            1_000_000, 0.0001);
    }

    /**
     * 检查消息是否已处理
     */
    public boolean isProcessed(String messageId) {
        return bloomFilter.mightContain(messageId);
    }

    /**
     * 标记消息已处理
     */
    public void markAsProcessed(String messageId) {
        bloomFilter.put(messageId);
    }
}
```

```java
@Service
public class CustomTracingService {

    @Autowired
    private RabbitTemplate rabbitTemplate;

    public void sendMessageWithTrace(Object message) {
        // 生成追踪 ID
        String traceId = UUID.randomUUID().toString();
        String spanId = UUID.randomUUID().toString().substring(0, 16);

        // 添加追踪信息到消息头
        Message msg = MessageBuilder
            .withBody(JSON.toJSONBytes(message))
            .setHeader("X-Trace-Id", traceId)
            .setHeader("X-Span-Id", spanId)
            .setHeader("X-Parent-Span-Id", "")
            .build();

        // 记录发送日志
        log.info("发送消息 - traceId: {}, spanId: {}", traceId, spanId);

        rabbitTemplate.convertAndSend("my.exchange", "routing.key", msg);
    }
}

// 消费者追踪
@RabbitListener(queues = "my.queue")
public void consumeWithTrace(Message message) {

    String spanId = message.getMessageProperties().getHeader("X-Span-Id");

    log.info("消费消息 - traceId: {}, spanId: {}", traceId, spanId);

    // 生成子 span
    String childSpanId = UUID.randomUUID().toString().substring(0, 16);
```

### 6.1 JSON 序列化

```java
@Configuration
public class RabbitMQConfig {

    @Bean
    public Jackson2JsonMessageConverter messageConverter() {
        ObjectMapper objectMapper = new ObjectMapper();
        objectMapper.registerModule(new JavaTimeModule());
        objectMapper.disable(SerializationFeature.WRITE_DATES_AS_TIMESTAMPS);
        objectMapper.setPropertyNamingStrategy(PropertyNamingStrategies.SNAKE_CASE);

        return new Jackson2JsonMessageConverter(objectMapper);
    }

    @Bean
    public RabbitTemplate rabbitTemplate(ConnectionFactory connectionFactory,
                                       Jackson2JsonMessageConverter messageConverter) {
        RabbitTemplate template = new RabbitTemplate(connectionFactory);
        template.setMessageConverter(messageConverter);
        return template;
    }
}
```

### 6.2 自定义序列化

```java
public class CustomMessageConverter implements MessageConverter {

    private final ObjectMapper objectMapper;

    public CustomMessageConverter() {
        this.objectMapper = new ObjectMapper();
    }

    @Override
    public Message toMessage(Object object, MessageProperties messageProperties) {
        try {
            byte[] bytes = objectMapper.writeValueAsBytes(object);
            messageProperties.setContentType("application/json");
            return new Message(bytes, messageProperties);
        } catch (JsonProcessingException e) {
            throw new MessageConversionException("序列化失败", e);
        }
    }

    @Override
    public Object fromMessage(Message message) throws MessageConversionException {
        try {
            String contentType = message.getMessageProperties().getContentType();
            if (contentType != null && contentType.contains("json")) {
                return objectMapper.readValue(message.getBody(), Map.class);
            }
            return new String(message.getBody());
        } catch (IOException e) {
            throw new MessageConversionException("反序列化失败", e);
        }
    }
}
```

## 七、消息压缩

### 7.1 Gzip 压缩

```java
@Component
public class MessageCompressor {

    /**
     * 压缩消息
     */
    public Message compress(Message message) throws IOException {
        byte[] compressed = compress(message.getBody());
        MessageProperties properties = message.getMessageProperties();
        properties.setContentEncoding("gzip");
        properties.setHeader("compressed", "true");
        return new Message(compressed, properties);
    }

    /**
     * 解压缩消息
     */
    public Message decompress(Message message) throws IOException {
        if (!"true".equals(message.getMessageProperties().getHeaders().get("compressed"))) {
            return message;
        }

        byte[] decompressed = decompress(message.getBody());
        MessageProperties properties = message.getMessageProperties();
        properties.setContentEncoding(null);
        properties.getHeaders().remove("compressed");
        return new Message(decompressed, properties);
    }

    private byte[] compress(byte[] data) throws IOException {
        ByteArrayOutputStream bos = new ByteArrayOutputStream(data.length);
        try (GZIPOutputStream gzip = new GZIPOutputStream(bos)) {
            gzip.write(data);
        }
        return bos.toByteArray();
    }

    private byte[] decompress(byte[] data) throws IOException {
        ByteArrayInputStream bis = new ByteArrayInputStream(data);
        ByteArrayOutputStream bos = new ByteArrayOutputStream();
        byte[] buffer = new byte[1024];
        int len;

        try (GZIPInputStream gzip = new GZIPInputStream(bis)) {
            while ((len = gzip.read(buffer)) > 0) {
                bos.write(buffer, 0, len);
            }
        }

        return bos.toByteArray();
    }
}
```

## 八、高级特性总结

| 特性 | 用途 | 适用场景 |
|------|------|----------|
| **死信队列** | 处理无法消费的消息 | 错误处理、告警 |
| **延迟队列** | 延迟处理消息 | 订单超时、定时任务 |
| **优先级队列** | 按优先级处理消息 | VIP 用户、紧急任务 |
| **消息去重** | 避免重复消费 | 幂等性要求 |
| **消息追踪** | 链路追踪 | 问题排查、性能分析 |
| **消息压缩** | 减少网络传输 | 大消息传输 |
| **消息转换** | 自定义序列化 | 特殊格式支持 |
